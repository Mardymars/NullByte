# NullByte

NullByte is an offline Android privacy utility for cleaning common metadata from photos, videos, and audio before sharing.

## Highlights

- Works locally on your device
- No account required
- No cloud processing
- No analytics
- No Internet permission
- No ads
- Creates clean copies instead of changing originals
- Supports batch picking and share-in from other apps
- Shows a before-and-after check for common location, device, time, and author details

## Supported Formats

- Photos and images
- MP4 video
- MOV video
- 3GP video
- M4A audio
- MP3 audio
- WAV audio

## Privacy Scope

NullByte checks for common metadata that is often shared with media files, including location, device, time, and author details. A verified clean result means those checked details were not found after cleaning.

Some file formats can store unusual hidden details, so sensitive files should still be reviewed before sharing.
